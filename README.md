# Iris Flower Classification

## Project Description
This project is a Machine Learning application that classifies Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

The model is built using the Random Forest Classifier from Scikit-learn and predicts the flower species based on sepal and petal measurements.

## Dataset
The project uses the famous Iris dataset containing the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- Random Forest Classifier

## Project Workflow
1. Load the Iris dataset.
2. Preprocess the data.
3. Split the dataset into training and testing sets.
4. Train the Random Forest model.
5. Predict the flower species.
6. Evaluate the model using accuracy score, confusion matrix, and classification report.
7. Visualize the results.

## Results
The model achieves high accuracy in classifying Iris flowers and provides a confusion matrix and classification report for evaluation.

## Files
- iris_classification.py
- Iris.csv
- README.md

## How to Run
1. Install the required libraries:
   ```
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
2. Place `Iris.csv` in the project folder.
3. Run:
   ```
   python iris_classification.py
   ```
